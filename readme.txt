# 이베스트 OPENAPI 출시 기념 거래량 이벤트 참여 매크로

## 원리
매일 10억을 거래하면 매일 2만원을 줌.
100만원 짜리 주식을 매일 500번 사고 팔면 됨(틱손실: 5원*500번=2500원)
제세공과금 22%를 제하면 매일 약 1.5만원 수익이 남.
(이벤트 신청계좌는 유관수수료까지 무료임. 근데 대면으로 계설한 계좌는 수수료 따로 나가는거 있어서 꼭 비대면 계좌인지 확인. 그냥 집에서 폰이나 컴으로 개설하면 비대면이고 증권사가서 파면 대면임)
10일부터 30일까지 영업일은 대략 총 15일 따라서 약 22.5만원의 순수익이 남.
이벤트 종료 후(5영업일 후) 받은 주식 상품권으로 아무 주식이나 사고 바로 팔아서 현금화하면 됨.

## 사용법
1. 이베스트 Open API 이벤트 참여 신청하기 (이베스트 홈페이지 -> 이벤트 -> OPEN API 거래이벤트)
2. 이베스트 api 키 발급 받기  (https://openapi.ebestsec.co.kr/howto-use)
3. 계좌에 150만원 넣기  
4. users.txt에 user.txt.example을 참고해서 계정 정보를 입력하기  
5. "python main.py 사용자이름"로 실행하기. (사용자이름은 users.txt에 설정한 것으로 입력.)
